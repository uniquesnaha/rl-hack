# DSAR-OpenEnv

This repository contains the DSAR hackathon project in [`dsar_env`](d:/rl-hack/dsar_env).

The deployable OpenEnv environment, inference script, Dockerfile, and submission assets all live in that folder so they stay together for Hugging Face Space deployment.

## Project Location

- Main project folder: [`dsar_env`](d:/rl-hack/dsar_env)
- Project documentation: [`dsar_env/README.md`](d:/rl-hack/dsar_env/README.md)
- OpenEnv manifest: [`dsar_env/openenv.yaml`](d:/rl-hack/dsar_env/openenv.yaml)
- Inference entrypoint: [`dsar_env/inference.py`](d:/rl-hack/dsar_env/inference.py)

## Notes

- The Hugging Face Space deployment uses the contents of `dsar_env`.
- This root README is here so the repository is readable from the GitHub landing page.
